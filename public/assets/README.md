# Assets Folder

Place the following image files in this directory:

## Required Images

1. **logo.png** (Main Logo)
   - Used in: WelcomePage navigation
   - Recommended size: 48x48px or higher
   - Format: PNG with transparency

2. **nova-logo.png** (Nova Logo)
   - Used in: RegisterPage navigation and hero section
   - Recommended size: 44x44px or higher
   - Format: PNG with transparency

3. **preview.png** (Platform Preview)
   - Used in: WelcomePage holographic frame
   - Recommended size: 380x420px or higher
   - Format: PNG or JPG

## Notes

- All images should be optimized for web (compressed)
- Use PNG format for logos (supports transparency)
- Use JPG or optimized PNG for preview images
- Maintain aspect ratios for best display
